The Approach

Four generators. One benchmark.

Gaussian Copula
Statistical baseline
Fits marginals per column, models correlations via multivariate Gaussian. Trains in seconds.
CTGAN
Conditional GAN
GAN — generator battles discriminator to produce realistic rows. 300 epochs.
TVAE
Variational autoencoder
Encodes rows into a latent space, decodes to produce synthetic samples. 300 epochs.
Tabular Diffusion
Custom DDPM — built from scratch
Reverses a 200-step Gaussian noising process. Custom 3.34M param MLP — built from scratch. 100 epochs.

All four trained in parallel on AWS SageMaker · ml.m5.4xlarge · outputs to S3

The Journey

Three rounds of
training to get it right.

Run 1 — Failed
Discriminator AUROC = 1.0 on all models — all data easily spotted as fake
Bug: regex=True default zeroed out 7 condition columns silently.
Bug: TVAE collapsed binary columns to zero — binary features misclassified as numerical.
Run 2 — Closer
Discriminator still 1.0 — but metrics improved after regex fix
Found: 90%+ of patients have zero inpatient visits — zero-inflated distributions are a hard ceiling for all generators.
Found: Synthetic mortality rate 14–16% vs real 19.5% — generators under-sample the minority class.
Run 3 — Best results
Applied all fixes + added Tabular Diffusion
Fix: log1p on 6 zero-inflated columns + post-hoc class rebalancing.
Result: Diffusion AUROC 0.9906 vs real 0.9926.

Takeaways

What we learned.

01
Diffusion models are in a different class. Correlation distance 0.503 vs 1.65–3.25 — not marginal, architectural.
02
Data quality dominated the first two runs. One wrong default (regex=True) zeroed 7 columns and invalidated everything. Engineering mattered more than model choice.
03
Synthetic-only training reaches 99.8% of real-data AUROC. Good enough to replace real EHR data for downstream ML — shareable with zero HIPAA risk.
